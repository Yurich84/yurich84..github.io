It's an app that allows people to lead a healthier lifestyle and choose healthier food. This project was on hold for several years, and when we started it again, the first thing we had to do was update all the frameworks and libraries. Eventually, we switched to Vue3 and began to use the composition API. We use MongoDB as a core DB because the data we took from an external source came in JSON and had a complex structure. A separate CRM system was developed for coaches, which interacted with a mobile application and an admin panel. To enhance user engagement, gamification elements were introduced. This involved creating a points accrual system, adding more interactive prompts, and developing several thematic browser games. As a result, the onboarding process was improved, and the overall user experience of the application was enhanced.
