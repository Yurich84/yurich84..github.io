This platform is for searching for vacancies and candidates. It not only automates unique recruiting processes but also allows candidates to participate in the creation of their resumes and search for vacancies. This work was entrusted to me and after some time I became a team lead in this direction. I gathered the team, set up the gitflow, CI/CD. Everything was built in docker, and even deployed with docker images on Digital Ocean. This was done in order to be able to flexibly scale it. We worked with Postgres and I set up full-text search there, added dictionaries, implemented fuzzy search, optimized and normalized the database. All this improved the matching of vacancies for candidates, which led to an increase in visitors to the site. Monthly attendance reached 100,000+.
